# نظام الترجمة التلقائية باستخدام Google Translate Snippet

## آلية العمل

### 1. الربط مع Google Translate
- تم دمج سكريبت `translate_a/element.js` من جوجل في ملف HTML
- عنصر `google_translate_element` مخفي تماماً باستخدام CSS
- لا يظهر أي عناصر من واجهة جوجل الافتراضية للمستخدم

### 2. التحكم البرمجي
عند اختيار لغة من القائمة المخصصة:
- يتم البحث عن عنصر `.goog-te-combo` (قائمة جوجل المخفية)
- يتم تغيير قيمته برمجياً إلى اللغة المختارة
- يتم إطلاق حدث `change` لتفعيل الترجمة فوراً
- عند العودة للعربية: يتم حذف الكوكيز وإعادة تحميل الصفحة

### 3. إخفاء واجهة جوجل
تم إضافة CSS لإخفاء:
- عنصر `google_translate_element`
- شريط `.skiptranslate`
- الإطارات `.goog-te-banner-frame`
- شعار جوجل `.goog-logo-link`

بالإضافة إلى كود JavaScript يعمل كل 100ms لضمان:
- إخفاء أي إطارات جديدة تظهر
- تصفير إزاحة `body.top` التي يفرضها جوجل

### 4. الرسائل الثابتة
تم إنشاء `staticMessages` dictionary يحتوي على:
- `paymentInstruction`: "توجه للكاشير للدفع"
- `itemAdded`: "تمت الإضافة"
- `itemRemoved`: "تم الحذف"
- `cartEmpty`: "السلة فارغة"

هذه الرسائل مترجمة يدوياً لجميع اللغات الـ15 ولا تعتمد على ترجمة جوجل.

### 5. ما يترجمه جوجل تلقائياً
- أسماء الأقسام (Categories)
- أسماء المنتجات (Products)
- أسماء الأحجام (Sizes)
- أي محتوى ديناميكي يُضاف لاحقاً

### 6. اللغات المدعومة
ar, en, zh, hi, id, ru, tl, yo, fr, tr, ms, sg, mn, fa, ja

## ملاحظات مهمة
1. **الاتصال بالإنترنت**: النظام يتطلب اتصال بالإنترنت ليعمل
2. **التحميل الأولي**: قد يستغرق 1-2 ثانية حتى يصبح جوجل جاهزاً
3. **العودة للعربية**: تتطلب إعادة تحميل الصفحة لضمان إزالة الترجمة تماماً
4. **المنتجات الجديدة**: ستُترجم تلقائياً دون الحاجة لإضافة أي كود إضافي
