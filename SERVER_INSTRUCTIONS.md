# تشغيل الخادم (Server)

## المتطلبات:
- Node.js مثبت على الجهاز
- npm (يأتي مع Node.js)

## الخطوات:

### 1. فتح Terminal في مجلد التطبيق:
```
cd c:\Users\HOME\Desktop\00\restaurant
```

### 2. تثبيت البرامج المطلوبة:
```
npm init -y
```

### 3. تشغيل الخادم:
```
node server.js
```

### 4. فتح الموقع:
- اذهب إلى: http://localhost:3000
- الآن جميع الأجهزة الموصولة بنفس الشبكة يمكنها الوصول للبيانات نفسها

## استخدام من أجهزة أخرى على نفس الشبكة:
- بدلاً من http://localhost:3000
- استخدم عنوان IP الجهاز:
- http://[IP_ADDRESS]:3000

### للحصول على عنوان IP للكمبيوتر:
اكتب في Terminal:
```
ipconfig
```

ابحث عن: IPv4 Address (مثل: 192.168.1.100)

ثم من جهاز آخر استخدم:
```
http://192.168.1.100:3000
```

## الخادم سيحفظ البيانات في:
- data.json (الملف الرئيسي)
- localStorage (كنسخة احتياطية)

## إذا حصل خطأ CORS:
الخادم مُعد بشكل صحيح ليقبل الطلبات من أي عنوان (CORS)
